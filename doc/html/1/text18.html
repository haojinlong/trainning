使用技巧二：协同commons-lang3使用

  • commons-lang3

    • 介绍：apache下的子项目，提供针对核心java类的一些额外的操作
    • 地址：http://commons.apache.org/proper/commons-lang/
    • 最新版本：3.3.2
    • 需要引入的jar包：commons-lang3-3.3.2.jar

  • 使用方式

    • 重写所有JavaBean的toString方法为：
      • return ToStringBuilder.reflectionToString(this);
    • debug时，直接传入相应的JavaBean即可
      • logger.debug(“the value of A: {}”, a);
